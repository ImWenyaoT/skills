# Verb standard

Strong action verbs carry a resume. The rule isn't "stuff in fancy verbs" — it's **lead each
bullet with a verb that names a real, strong action**, and kill the empty ones.

## Strong action verbs by category

- **Creation:** designed · built · implemented · developed · engineered · architected · shipped · produced
- **Research / analysis:** validated · analyzed · evaluated · audited · diagnosed · investigated · profiled
- **Improvement:** optimized · reduced · hardened · refactored · streamlined · stabilized · automated
- **Leadership:** led · drove · spearheaded — ⚠️ use sparingly on solo/student projects (it rings
  false); for solo work prefer "independently designed/built".
- **Collaboration:** collaborated · co-built · supported · integrated
- **Delivery:** delivered · launched · released · merged (to mainline)

## Banned filler verbs (weak or zero-information)

**responsible for · involved in · participated in · worked on · helped with · handled · dealt with · assisted**

Before finalizing, grep the resume for these — the count should be **0**.

## Usage notes

- Don't repeat a verb **within one entry**; core verbs (design/implement/build) naturally recur
  across different projects — that's fine, just keep overall variety (validate/harden/automate/
  refactor/integrate all in play). The thing to eliminate is filler verbs, not legitimate recurrence.
- Match verb category to the selling point: quality-gate / eval work uses Research/Improvement verbs
  (validated/hardened), not all Creation verbs (designed/built).
- One strong verb leads the clause; don't let two verbs fight, and don't dilute it with a
  prepositional "turned … into …" construction.

## Non-English resumes: the back-translation check

If you write in a language other than English (e.g. Chinese), don't transliterate English verbs.
Instead: **write a native strong verb → mentally translate it to English → is it on the strong-verb
list above?** If it comes out weak or has no equivalent, swap it. This keeps the resume idiomatic
while holding the same strong-verb bar.
