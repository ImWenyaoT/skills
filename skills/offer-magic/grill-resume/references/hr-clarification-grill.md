# HR Clarification Grill

Use a bounded grill when an HR finding or resume decision depends on information that cannot be verified from the resume, JD, profile, or evidence sources.

## Trigger the grill

Grill only when at least one condition holds:

- two plausible interpretations would change advance / hold / reject, content priority, or claim wording;
- a material HR confusion depends on an unknown candidate fact, contribution boundary, target constraint, or preference;
- accepting or rejecting feedback would otherwise require guessing.

Skip the grill when the answer is already discoverable, the unknown can safely remain omitted, or the answer would not change the action.

## Resolve one branch at a time

1. Search the available evidence before asking the user.
2. Rank unresolved questions by how much they can change the decision.
3. Ask only the highest-leverage question and wait for its answer.
4. State why the answer matters. For a decision, recommend one option and explain the trade-off. For a factual question, recommend an answer format, never the missing fact.
5. Record the answer as a user statement, not independent verification. Ask the next question only if it remains material after the new answer.

Pause only the affected claim or decision; continue independent work. Stop when the remaining unknowns cannot change the recommendation, the user asks to stop, or the next question would merely improve polish.

## Preserve boundaries

- Ask for facts, decisions, or examples—not agreement with the reviewer.
- Do not lead the user toward a stronger claim.
- Keep uncertainty explicit when the user is unsure.
- Rebuild the frozen packet and use a fresh HR review when new information materially changes the candidate version or review context.
